# Day 7

## What I Built

Expanded Investor OS with additional technical indicators.

Completed:

- 20-Day Moving Average
- Standard Deviation
- Bollinger Bands
- MACD
- Signal Line
- MACD Histogram
- Enhanced Technical Analysis Dashboard

The dashboard now combines trend, momentum, volatility, and volume indicators into a single visualization.

## What I Learned

Rolling windows can be reused to build more advanced indicators.

Many technical indicators are built from previously calculated values instead of raw prices.

Visualization is just as important as calculation because it makes patterns easier to recognize.

## Challenges

Initially the Bollinger Bands appeared missing because only one month of data was used. After testing with six months of data, the bands displayed correctly.

Also learned the importance of activating the virtual environment after opening a new terminal.

## Reflection

Investor OS is beginning to resemble a real market analysis platform instead of a collection of small Python exercises.

The project feels much more organized, and adding new indicators is becoming easier because the codebase is modular.
