
# Day 1 — Stock Data Pipeline

## Objective

Learn the basics of Python by building a program that fetches live stock market data.

## What I learned

- Created my first standalone Python project.
- Learned why virtual environments are important.
- Created and activated a `.venv`.
- Installed packages using `pip`.
- Imported external libraries (`yfinance` and `pandas`).
- Downloaded live stock market data.
- Accepted user input using `input()`.
- Used variables to store values.
- Extracted data from a Pandas DataFrame.
- Used `.iloc`, `.max()`, and `.min()`.
- Formatted prices using f-strings (`:.2f`).

## Bugs encountered

PowerShell initially blocked virtual environment activation due to the execution policy.

Fixed using:

```
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
```

Later I received:

```
NameError: close_price is not defined
```

I learned that variables must be created before they can be used.

## Reflection

Today I fetched real-time stock prices for companies like NVDA and MSFT using Python.

Seeing live financial data appear from my own program was the first time my code interacted with the real world.

I also became more comfortable reading error messages instead of fearing them.
